# Invention – Bootstrap Website

A beginner-friendly Bootstrap 5 project that showcases great scientific inventions
with a clean UI, responsive layout, and dark theme.

## 🔹 Features
- Responsive Navbar with dropdown
- Bootstrap 5 grid system
- Dark mode theme
- Hero section with call-to-action buttons
- Clean and structured layout

## 🔹 Tech Stack
- HTML5
- CSS3
- Bootstrap 5

## 🔹 Purpose
This project was built while learning Bootstrap and responsive web design.
It focuses on understanding layout, components, and real-world UI structure.
